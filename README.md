# Amazola Sweet Lounge 🌿

The official website for **Amazola Sweet Lounge** — a cozy café and restaurant in Kabuku, Limuru, Kenya. Every meal served contributes to the **Fountain of Hope Rescue Centre**, a children's home on NGECHA-KABUKU Rd, Limuru.

## Pages

| Page | Description |
|---|---|
| Home | Hero, featured menu highlights & brand story |
| Menu | Full food & drinks menu with categories |
| Reservations | Table booking form |
| Fountain of Hope | About the rescue centre, impact stats & donation panel |
| Contact | Location map, contact details & enquiry form |

## Tech Stack

- **React** + **TypeScript** — component-based UI
- **Vite** — fast dev server & bundler
- **Tailwind CSS** — utility-first styling
- **shadcn/ui** — accessible UI components
- **Framer Motion** — animations & transitions
- **Vercel** — deployment

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

> Requires Node.js 18+

## Project Structure

```
src/
├── pages/       # Route-level page components
├── components/  # Shared & UI components
├── assets/      # Images
├── hooks/       # Custom React hooks
└── lib/         # Utilities
```
